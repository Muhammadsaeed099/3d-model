Methodology & honesty note

What's sourced, what's rounded, and what's invented — so nothing here pretends to be more precise than it is.

About the numbers

Sales figures are approximate lifetime units sold, rounded to the nearest million, compiled from publicly reported publisher figures and aggregated sales trackers. These numbers drift upward as catalogue titles keep selling, so treat them as a ballpark snapshot (cutoff around 2024–2025), not an audited ledger. Strict ranking order is preserved with no ties.

Sourced vs invented

  • Sales & ranking — sourced and rounded from public reporting.
  • Cover cards — original stylised illustration generated in-app. These are not official box art or scans; they're abstract placeholders so the project ships clean of copyrighted artwork.
  • Flags — decorative cue for the publisher's home region, drawn procedurally.
  • Podium height — scaled linearly to the sales number.
  • Per-title caveats (bundling, edition scope, etc.) appear in each exhibit's detail panel.

Running it locally

This is one self-contained HTML file — double-click to open in current Chrome or Firefox, no server, login, or paid service. The only network fetches are Three.js and the web fonts; for a fully offline build, drop those into the project folder and point to them locally. To swap any game, edit the GAMES list at the top of the script (it mirrors a data/games.json record shape) and reload — the layout, filters, tour, and panels rebuild themselves automatically.
